<objective>
Создать персональное хранилище знаний для нового пользователя.
Ядро системы — три слоя понимания пользователя: raw-thoughts (дословный голос), insights (структурированные мысли с цитатами), thinking (эволюция мысли с дельтами).
</objective>

<process>

## Шаг 1: Узнать пользователя

Спроси (НЕ пропускай, НЕ угадывай, ЖДИ ответ):

1. **Кто ты?** (роль, профессия, чем занимаешься)
2. **Зачем тебе хранилище?** (учёба, работа, исследования, саморазвитие, всё вместе?)
3. **Какие источники будешь загружать?** (статьи, видео, подкасты, книги, свои заметки, рабочие документы?)
4. **Как хочешь отслеживать эволюцию мышления?**
   - **По темам** — мысль растёт вокруг темы
   - **По вопросам** — мысль растёт вокруг конкретного вопроса
   - **По проектам** — мысль привязана к делу
   - **Смешанно** — по ситуации

## Шаг 2: Предложить структуру

**Обязательные папки** (создаются всегда, не спрашивай):
- `wiki/raw-thoughts/`, `wiki/insights/`, `wiki/thinking/`

**Рекомендуемые** (предложи по профилю, пользователь выбирает):
- Для всех: `wiki/concepts/`
- Учёба: `wiki/courses/`, `reports/`
- Работа: `wiki/projects/`, `wiki/decisions/`
- Исследования: `wiki/people/`, `wiki/references/`
- Саморазвитие: `wiki/practices/`, `wiki/people/`

Покажи список, спроси что убрать/добавить. ЖДИ ответ.

## Шаг 3: Создать структуру

### 3.1. Папки
Все папки с `.gitkeep`. Обязательно: `raw/`, `wiki/raw-thoughts/`, `wiki/insights/`, `wiki/thinking/`, `wiki/_templates/` + выбранные пользователем.

### 3.2. Шаблоны
Прочитай файлы из `.claude/skills/init-vault/templates/` и скопируй в `wiki/_templates/` хранилища:

**Обязательные (всегда):**
- `templates/raw-thought.md` → `wiki/_templates/raw-thought.md`
- `templates/insight.md` → `wiki/_templates/insight.md`
- Шаблон thinking — выбери по ответу на вопрос 4:
  - «по темам» → `templates/thinking-by-topic.md`
  - «по вопросам» → `templates/thinking-by-question.md`
  - «по проектам» → `templates/thinking-by-project.md`
  - «смешанно» → `templates/thinking-mixed.md`
  - Скопируй выбранный как `wiki/_templates/thinking.md`

**По выбору пользователя:**
- `wiki/concepts/` → скопируй `templates/concept.md`
- `wiki/people/` → скопируй `templates/person.md`
- `wiki/practices/` → скопируй `templates/practice.md`
- Для других папок — создай минимальный шаблон (frontmatter + 2-3 секции).

### 3.3. index.md и log.md
- `index.md` — заголовок + пустой каталог
- `log.md` — запись о создании с датой, профилем пользователя и списком папок

### 3.4. CLAUDE.md
Прочитай `.claude/skills/init-vault/claude-md-base.md` как каркас.
Адаптируй под пользователя:
- Заполни секцию «Структура» реальными папками
- В «Эволюция мысли» оставь ТОЛЬКО вариант, соответствующий выбору
- Если учёба — добавь операцию INGEST REPORT
- Если работа — добавь секцию про решения и контекст проектов
- Если саморазвитие — добавь уровни резонанса в инсайты

## Шаг 4: Подтверждение
Покажи список созданных файлов и папок. Объясни как начать работу.

</process>

<rules>
- НИКОГДА не пропускай шаг 1 и шаг 2
- Три папки понимания (raw-thoughts, insights, thinking) — ОБЯЗАТЕЛЬНЫ
- Шаблоны копируй из `templates/`, не пиши заново
- CLAUDE.md генерируй на основе `claude-md-base.md`
- Все файлы на русском
- НЕ создавай примеры страниц — только структуру, шаблоны и инструкцию
</rules>
